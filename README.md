# Front End
To install this repo on vs code

step:-

1. Install vs. code community version.

2. Select "Node Js"

3. Clone the repository

```bash

git clone git@github.com:itax-easy/itax-easy-next.git

```

  

5. Install packages

```bash
# Using npm
npm install

# Using yarn
yarn install

# Using pnpm
pnpm install

# Using bun
bun install
```

